Employee Web App (Embedded Tomcat)
---------------------------------
How to run:
1. Open this folder as a Maven project in IntelliJ IDEA.
2. Ensure JDK 17+ is configured.
3. Build the project (Maven).
4. Run the main class: com.example.employee.StartServer
5. Open Chrome at: http://localhost:8080/employee.html
